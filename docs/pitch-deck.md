# Soso DNA — Pitch Deck Outline

## 1. Problem

SoDEX traders lose edge not from bad direction, but from **execution fragility** — entering during spread expansion, liquidity thinning, and post-loss emotional patterns. Raw PNL hides the forensic story.

## 2. Solution

**Soso DNA** turns wallet-connected SoDEX history into behavioral quant intelligence:

- Trading DNA profile (liquidity stress, revenge probability, execution precision)
- Forensic trade replay with orderbook-derived frames
- SoSoValue macro/news context overlaid on regime analysis
- Pre-trade risk engine with intervention modal
- Live AI reasoning stream bound to evidence

## 3. How It Works

```
ValueChain Wallet → Auth → SoDEX Sync → SoSoValue Context → Features → Risk + AI → Dashboard
```

## 4. Integrations (Buildathon)

- **Required:** SoSoValue API — news, macro, pair context in UI and risk scoring
- **Bonus:** SoDEX testnet — trades, account state, orderbook snapshots
- **Bonus:** AI + risk — WebSocket forensic stream, no automatic live trading

## 5. Demo Narrative (90s)

1. Land on cinematic homepage — "behavioral forensics for SoDEX"
2. Connect ValueChain testnet wallet, sign message
3. Dashboard populates with real imported trades
4. Open losing trade replay — spread expansion annotated
5. Market Regime — live SoSoValue macro headline
6. Risk Engine — simulate entry, trigger intervention at high similarity
7. AI panel — streaming phases with cited spread + macro evidence

## 6. Market

Active perp traders on SoDEX/ValueChain ecosystem who want institutional-grade post-trade analysis without giving up custody.
